## 🎯 清理原则

### 保留标准

1. **唯一性** - 内容不重复
2. **实用性** - 提供可操作的指导
3. **时效性** - 反映当前状态
4. **完整性** - 信息完整准确

### 归档标准

1. **重复内容** - 已有更好的版本
2. **过时信息** - 不再适用
3. **临时文档** - 问题已解决
4. **非核心功能** - 暂时不需要
