## 🚀 Quick Start (3 Steps)

### Step 1: Install Everything
```bash
npm install
cd frontend && npm install && cd ..
```

### Step 2: Deploy Contracts
```bash
# Start local blockchain
npx hardhat node

# In another terminal, deploy
npm run deploy:local
```

### Step 3: Start Frontend
```bash
cd frontend
npm run dev
```

**Visit: http://localhost:3000** 🎉
